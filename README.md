# webtoons

## 웹툰 모아보기 사이트 개발중
포트폴리오용 웹툰 모아보기 사이트 개발중입니다.
여러 사이트 웹툰을 모아서 관리하고 등록된 웹툰들 중에서 아직 보지 않은 웹툰들을 알려줍니다.


## 구현

로그인, 회원가입
오늘의 웹툰 슬라이드
인기웹툰 슬라이드
웹툰 등록 모달

## 구현중

아직 보지않은 웹툰
전체 웹툰 한눈에 보기
